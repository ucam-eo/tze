    Class MaplibreTesseraManager

    MapLibre display wrapper around a SourceManager.

    Remarks

    Manages per-zone MaplibreTesseraSource display instances, routing display operations to the correct zone(s). Data-access operations (chunk loading, embedding queries, zone routing) are available on manager.
