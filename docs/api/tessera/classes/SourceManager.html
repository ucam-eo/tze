    Class SourceManager

    Multi-zone routing manager for TESSERA Zarr stores.

    Remarks

    Manages a set of ZoneDescriptors, each representing one UTM zone with its own Zarr store. Sources are opened lazily on first access and cached. Events from per-zone sources are forwarded to manager listeners, so callers can subscribe once on the manager rather than on each zone.
