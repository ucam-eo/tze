    6-element affine transform mapping pixel indices to UTM coordinates.

    Remarks

    GDAL GeoTransform layout: [pixelW, 0, originX, 0, -pixelH, originY]. Pixel (col, row) maps to UTM (originX + col*pixelW, originY - row*pixelH), where originX = t[2], pixelW = t[0], originY = t[5], pixelH = -t[4].
