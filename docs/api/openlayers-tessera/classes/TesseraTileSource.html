    Class TesseraTileSource

    OpenLayers tile source that renders TESSERA RGB tiles from a Zarr pyramid.

    Remarks

    Uses TesseraTileRenderer to fetch and render tiles. Integrates with OpenLayers' XYZ source via a custom tile loader function.

    Example

    import TileLayer from 'ol/layer/Tile';
    const source = new TesseraTileSource({
      url: 'https://dl2.geotessera.org/zarr/v1/2025.zarr/global_rgb',
      variable: 'rgb',
    });
    const layer = new TileLayer({ source });
    map.addLayer(layer);
